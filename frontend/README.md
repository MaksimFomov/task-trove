# TaskTrove Frontend

React frontend приложение для платформы управления задачами TaskTrove.

## Технологии

- **React 18** - UI библиотека
- **TypeScript** - типизация
- **Vite** - сборщик и dev сервер
- **React Router** - маршрутизация
- **TanStack Query** - управление состоянием сервера
- **Zustand** - управление состоянием клиента
- **Axios** - HTTP клиент
- **Tailwind CSS** - стилизация
- **React Hot Toast** - уведомления
- **Lucide React** - иконки
- **WebSocket** - чат в реальном времени

## Установка

```bash
cd frontend
npm install
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:3000`

## Сборка для production

```bash
npm run build
```

## Структура проекта

```
frontend/
├── src/
│   ├── components/      # Переиспользуемые компоненты
│   ├── pages/          # Страницы приложения
│   │   ├── customer/   # Страницы для заказчиков
│   │   ├── performer/  # Страницы для исполнителей
│   │   └── admin/      # Страницы для администраторов
│   ├── services/       # API сервисы и WebSocket
│   ├── store/          # Zustand store
│   ├── types/          # TypeScript типы
│   ├── utils/          # Утилиты
│   └── App.tsx         # Главный компонент
├── index.html
├── package.json
└── vite.config.ts
```

## Основные функции

### Для заказчиков (Customer)
- Создание и управление заказами
- Просмотр откликов от исполнителей
- Утверждение исполнителей
- Общение через чат
- Отправка технических заданий
- Оставление отзывов

### Для исполнителей (Performer)
- Просмотр доступных заказов
- Отклик на заказы
- Управление своими откликами
- Обновление портфолио
- Общение через чат

### Для администраторов (Administrator)
- Управление пользователями
- Активация/деактивация пользователей
- Просмотр портфолио пользователей

## API Endpoints

Frontend взаимодействует с backend API на `http://localhost:8080/api`

Все запросы автоматически включают JWT токен из localStorage.

## WebSocket

Чат использует WebSocket подключение к `ws://localhost:8080/ws/chat/{chatId}`

## Переменные окружения

Настройки прокси для API находятся в `vite.config.ts`
